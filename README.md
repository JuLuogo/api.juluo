## 🚀 快速开始

欢迎来到橘络人物分析文档！这是一个全面的分析项目，包含橘络在多个群聊和私人聊天中的互动分析。

### 主要功能
- 📊 **综合报告** - 完整的人物互动关系分析
- 💬 **群聊分析** - DPの交流群和基友交流群的详细分析
- 🔒 **私人聊天** - 深度私人聊天内容分析
- 📈 **分析维度** - 互动模式、情绪态度、主题关键词
- 💕 **关系分析** - 橘络与各人物的关系深度分析

### 导航方式
1. **侧边栏导航** - 使用左侧导航栏浏览所有文档
2. **搜索功能** - 在顶部搜索框快速查找内容
3. **页面导航** - 使用页面底部的上一页/下一页按钮
