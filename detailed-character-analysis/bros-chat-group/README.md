### 1. 社交导向型交流
- 主要进行社交互动

- 交流风格轻松活泼

### 2. 高活跃度
- 成员互动频繁
- 消息数量庞大
- 群聊氛围活跃

### 3. 亲密关系

- 情感表达丰富
- 形成紧密的社交圈

### 核心互动网络
- **掐咯一** 与 **橘络**：双核心互动关系

- **掐咯一/橘络** 与 **通宵倒问**：时段互补互动

### 群聊氛围特点
- 活跃度极高
- 交流风格轻松活泼
- 话题多样
- 成员间关系密切
